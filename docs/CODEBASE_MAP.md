# Codebase Map

- `src/core/`: agent loop, configuration, provider orchestration.
- `src/tools/`: execution/file/browser and supporting utility tools.
- `src/health/`: circuit breaker and self-healing support.
- `src/memory/`: persistence and retrieval logic.
- `src/ui/server.js`: WebUI API and websocket event flow.
- `scripts/`: initialization, git sync, health checks, research tasks.
- `tests/`: smoke tests and WebUI e2e coverage.
